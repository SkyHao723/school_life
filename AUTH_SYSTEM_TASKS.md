## 宝塔面板部署提示

根据您提供的信息，您的服务器使用宝塔面板管理，这将大大简化部署过程：

1. 可以通过宝塔面板的图形界面轻松上传文件
2. 站点配置和反向代理设置更加直观
3. SSL证书管理更为便捷
4. 可以通过宝塔面板直接监控应用运行状态

## CentOS系统部署提示

您的服务器运行的是基于Red Hat的系统（如CentOS或Alibaba Cloud Linux）：

1. 不能使用apt或Debian相关的命令
2. 应使用yum或dnf包管理器
3. Node.js安装需要使用适用于RHEL系统的安装方法
4. 已在部署指南中提供具体安装步骤
